# 关于 Bitsy

> 嗨！Bitsy 是一个用于制作小型游戏或小型世界的小编辑器。其目标是让制作能够四处走动、与人交谈并置身于某个地方的游戏变得简单。
>
> 祝你用 Bitsy 创作愉快！
>
> &ndash; <cite>[adam](https://www.twitter.com/adamledoux)</cite>
